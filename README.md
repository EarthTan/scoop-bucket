# EarthTan's Scoop Bucket

This is a [Scoop](https://scoop.sh/) bucket for Windows command-line tools by
[@EarthTan](https://github.com/EarthTan).

## Available manifests

| Manifest | Description |
|----------|-------------|
| [`pano-viewer`](bucket/pano-viewer.json) | A 360° panoramic image viewer written in pure Rust. |

## Install

```powershell
scoop bucket add EarthTan https://github.com/EarthTan/scoop-bucket
scoop install EarthTan/pano-viewer
```
